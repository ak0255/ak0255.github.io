差分约束

求解不等式组的可行解或者最值

前提条件：从源点出发可以遍历所有边

1、可行解

  • 根据大小关系建图，例如v <= u + w，转换为从u到v有一条长度为w的边

  • 从源点跑单源最短路，如果存在负环，无可行解，否则dist[i]即为一个可行解

2、最值

  • 最小值，通过v >= u + w建立从u到v有一条长度为w的边，跑最长路，dist[i]即为i的最小值
  • 最大值，通过v <= u + w建立从u到v有一条长度为w的边，跑最短路，dist[i]即为i的最大值

特殊情况x >= c，这种情况，应当建立虚拟源点x0，得到x >= x0 + c
